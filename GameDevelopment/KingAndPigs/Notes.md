# Project Overview: Kings and Pigs

This project is based on the [Kings and Pigs](https://github.com/chriscourses/kings-and-pigs) tutorial by [Chris Courses](https://www.youtube.com/watch?v=Lcdc2v-9PjA). This repository serves as a personal documentation space to track the development process and technical insights gained during the build.

## Tools and Implementation
The game world is built using [Tiled](https://www.mapeditor.org/). Beyond simply using the provided assets, the project focuses on the logic required to extract and implement map data effectively. To handle environmental animations, such as interactive doors, the [GSAP](https://cdnjs.com/libraries/gsap) library is used to ensure fluid transitions.

The core of the program is built on Object-Oriented Programming (OOP) principles, which provide the necessary structure for managing game entities, physics, and state changes.

## Learning Objectives
The goal of this project is to develop the specific mindset required for game development. Key focus areas include:

* Architectural Logic: Understanding how to translate game ideas into scalable code.
* OOP Proficiency: Applying object-oriented patterns to real-time interactions and collision handling.
* Independent Research: Encouraging experimentation and problem-solving beyond the tutorial’s scope to deepen technical knowledge.

This project serves as a practical foundation for building more complex game systems and mastering the fundamentals of interactive development.
